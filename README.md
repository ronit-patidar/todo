# Todo App

A simple and responsive Todo application built with React. This app allows users to manage their daily tasks efficiently.

## Demo
>[Link](https://lottery-game-eosin-eight.vercel.app/)
## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Convert all tasks to uppercase
* Responsive and user-friendly interface

## Technologies Used

* React
* JavaScript (ES6+)
* CSS
* UUID

## Installation

1. Clone the repository:

```bash
git clone <[repository](https://github.com/ronit-patidar/todo/)>
```

2. Navigate to the project directory:

```bash
cd todo-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## Project Structure

```text
src/
```

## Usage

* Enter a task in the input field.
* Click **Add Task** to add it to the list.
* Use the available buttons to update or delete tasks.
* Manage your daily activities efficiently.
